# Запуск клиентской части
### Настройте конфигурационный файл 
1. Перейдите в папку gui и откройте файл config.json
2. host - Ip-адрес до вашего API-сервера
3. port - Порт подключения к вашему API-серверу
4. token - Токен доступа к запросам
### Разверните виртуальное окружение в проекте
Зайдите в терминал в папке app и введите следующую команду
``` bash
python -m venv venv
```
После успешного создания виртуального окружения необходимо его активировать командой
``` bash
venv/Scripts/activate
```
Скачайте все необходимые для проекта библиотеки, используя команду
``` bash
pip install -r requirements.txt
```
### Запуска исполняемого файла
Для запуска главного файла maon.py необходимо использовать аргументы во время запуска
Список аргументов:
1. --print - Вывод информации (сахар, инсулин, еда, устройства) в виде таблиц
2. --printLoop - Вывод информации в виде таблиц в цикле (бесконечно)
3. --graphD - Вывод графика сахар за 1 день, необходимо в файле main.py изменить дату для начала графика и его конца
4. --console - Запустить режим работы с программой через консоль
5. --info - Вывести таблицу всех аргументов и их описание

Пример запуска:
``` bash
python main.py --printLoop  # Запуск вывода информации в цикле 
python main.py --console  # Запуск работы с программой через консоль
python main.py --graphD  # Запуск графика сахаров за 1 день
```
