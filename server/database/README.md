# Структура Базы Данных
``` database
database
├── Sugar (Table)
│   ├── id: int (Уникальный идентификатор записи)
│   ├── date: int (unix дата записи)
│   ├── sugar: float (Значение сахара)
│   ├── tendency: str (Тенденция поведения сахара)
│   └── difference: str (Разница сахара с предыдущей записью)
├── Insulin (Table)
│   ├── id: int (Уникальный идентификатор записи)
│   ├── date: int (unix дата записи)
│   ├── insulin: float (Значение поданого инсулина)
│   ├── carbs: float (Значение съеденых углеводов)
│   ├── duration: int (Значение времени действия записи)
│   └── type: str (Тип инсулино-терапии)
└── Device (Table)
    ├── id: int (Уникальный идентификатор записи)
    ├── date: int (unix дата записи)
    ├── phone_battery: int (Заряд батареи телефона)
    ├── transmitter_battery: int (Заряд батареи трансмиттера)
    ├── pump_battery: int (Заряд батареи помпы)
    ├── pump_cartridge: float (Кол-во инсулина в резервуаре)
    ├── cannula: int (unix дата установки катетора)
    ├── insulin: int (unix дата установки инсулина)
    ├── sensor: int (unix дата установки НМГ)
    ├── pump_model: str (Модель помпы)
    ├── phone_model: str (Модель телефона)
    └── transmitter_model: str (Модель трансмиттера)
```
